# not-redis
A lowly implemented redis clone

## How to run
Requires Python3

```
python3 server.py
```

## Items Implemented:

 * incr
 * incrby
 * get
 * set
 * del(ete)
 * exists
 * lpush
 * lpop
 * llen
 * lrem
 * lset
 * zadd
 * zrem
 * zcard
 * zcount
 * zrank
 * zrange is semi-implemented (unfinished)
